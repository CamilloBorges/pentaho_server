# Other Configurations

Este diretório contém configurações diversas do Pentaho e Tomcat.

## Arquivos de Configuração

### Pentaho System
- `pentaho-solutions/system/pentaho.xml` - Configuração principal
- `pentaho-solutions/system/defaultUser.spring.properties` - Usuário padrão
- `pentaho-solutions/system/security.properties` - Propriedades de segurança

### Tomcat
- `tomcat/bin/startup.sh` - Script de inicialização
- `tomcat/webapps/pentaho/WEB-INF/web.xml` - Web application descriptor
